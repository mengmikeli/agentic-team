## Metrics Guide

Capture at sprint close (sprint-ops close):
- **Commits / PRs** — always available from git
- **Duration** — calendar days from start to close
- **Model** — execution model used (subagent swarm, multi-agent team, hybrid, solo)
- **Token usage** — if pew or session logs available, add as a note in the sprint summary

Keep it lightweight. Don't force metrics you can't easily gather.
